Quick Start

    GETTING STARTED

  1. Don't forget to read the README file.
  2. Avoid the mouse. Use keyboard shortcuts to open the Chrome Inspector and when editing code in Atom.

    3. THE CONSOLE

  3. Use console.log to emit secret messages into the Chrome Console tab.
  4. Also: console.info, console.warning, and console.error

    5. MODIFYING THE DOM

  5. Use the jQuery() function to select elements on the page.
  6. jQuery-selected elements have functions that allow you to change their content.
  7. For example, use the .html() function to change the HTML content of an element.

    8. EVENTS

  8. Events happen all the time as the user browses a web page.
  9. Use the .on() function to start "listening" for an event on a DOM element or a set of elements.

Cheat Sheet for Week 5

  • Use let or const to define variables.
  • Define functions by assigning them to variable names, like this: 
    
  var f = function(arg1, arg2) {
    // code goes here
  }
  • We can use the for (let item of list) construct to "loop" through a list of items.
  • Given an element elem, you can attach a click event handler with:
    • elem.on("click", myFunctionName)
  • Event handler functions should accept one argument, the event object:
    • 
  var myHandler = function(event) { ... }
  • To prevent the browser from also responding to a click event, you can call event.preventDefault();
  • When inside an event handler function, you can refer to the original element by using jQuery(this).
  • To reserve a blank section of the page for your JavaScript to add HTML later, use an empty div and give it an id attribute so your code can find it: 
    
  <div id="mydiv"></div>
  • Generate a random integer from 1 to 10 with: Math.floor(Math.random() * 10) + 1
